⚙ Configurações

Configure notificações e o intervalo de consulta ao DataJud.

📧 Email (SMTP)

Ex: smtp.gmail.com, smtp.office365.com
Para Gmail, use uma senha de app. Deixe em branco para manter a atual.

✈ Telegram (opcional)

Crie um bot em @BotFather no Telegram. Deixe em branco para manter o atual.
Use @userinfobot para descobrir seu chat_id

🔑 API DataJud

Chave pública do CNJ. Obtenha em datajud-wiki.cnj.jus.br. Se vazio, usa a chave padrão.

⏱ Consulta Automática

A cada quantas horas o sistema consulta o DataJud por novos andamentos

📖 Como usar

  1. Importe seus processos: Exporte do Astrea (Administração > Escritório > Manutenção de conta) e faça upload do XLSX no Dashboard
  2. Cadastre manualmente processos que não estão no Astrea usando o botão "Novo Processo"
  3. Atualize manualmente cada processo com o botão 🔄 ou use "Atualizar Todos" para consultar o DataJud
  4. Configure notificações acima para receber alertas de novas movimentações por email ou Telegram
  5. O sistema atualiza automaticamente no intervalo configurado
